# password-maker

構成をカスタマイズして複数の強力なパスワードを生成できるウェブベースのツールです。最大の特徴は、各文字に日本語の読み（ヨミガナ）を表示する機能であり、パスワードを読みやすくし、口頭で伝えやすくします。

## デモ

[**ライブデモ**](https://code4fukui.github.io/password-maker/)

ユーザーインターフェースでは、数字、アルファベット大文字、アルファベット小文字、記号の桁数を設定してパスワードの構成を調整できます。また、生成するパスワードの総数を指定してから「パスワードを生成する」ボタンをクリックします。

## 機能

- **構成のカスタマイズ:** 各パスワードの数字、アルファベット大文字、アルファベット小文字、記号の正確な桁数を指定できます。
- **一括生成:** 一度に大量のパスワードを生成できます（デフォルトは200個）。
- **ルビ表示（ヨミガナ）:** 生成されたパスワードの各文字には、HTMLのルビタグ（`<ruby>`）を使用して日本語の読みが表示されるため、口頭で簡単に伝えることができます。

## 使い方

1. [ライブデモ](https://code4fukui.github.io/password-maker/)を開きます。
2. 数値入力フィールドを使用して、各文字タイプ（数字、アルファベット大文字など）の桁数を調整します。合計桁数は自動的に更新されます。
3. 「パスワード生成数」フィールドに、生成したいパスワードの数を設定します。
4. **パスワードを生成する**ボタンをクリックします。
5. 生成されたパスワードのリストがページ上に表示されます。

## 技術詳細

このツールは自己完結型のHTMLファイルであり、ビルドプロセスは必要ありません。

- **依存関係:** `js.sabae.cc` の2つの外部JavaScriptモジュールを利用しています。
  - [CSV.js](https://js.sabae.cc/CSV.js): 文字データの解析用。
  - [rnd.js](https://js.sabae.cc/rnd.js): ランダムな文字選択用。
- **データソース:** 文字と読みの対応付けは、ローカルのCSVファイル（`num_d-yomi.csv`、`symbol-yomi.csv`など）から読み込まれます。これらはシンプルな `char,yomi` 形式（例: `-,ハイフン`）に従っています。
